# Nationality Detection System

This project predicts the nationality and emotion of a person from an uploaded image using Machine Learning and Computer Vision.

Features:
- Predicts nationality
- Detects emotions
- For Indian individuals:
  - Predicts age
  - Predicts dress colour
  - Detects emotions
- For United States individuals:
  - Predicts age
  - Detects emotions
- For African individuals:
  - Predicts dress colour
  - Detects emotions
- For other nationalities:
  - Predicts nationality and emotions only
- GUI with image preview
- Output section for displaying results

Technologies Used:
- HTML
- CSS
- JavaScript
- TensorFlow.js
- OpenCV
